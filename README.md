# Numerical Base Converter 🔢
A collection of Python scripts that allow you to convert Decimal numbers into various bases (Binary, Hexadecimal, Octal, and custom bases up to 36).

## 🚀 Features
* **Decimal to Binary:** Converts base 10 to base 2.
* **Decimal to Octal:** Converts base 10 to base 8.
* **Decimal to Hexadecimal:** Converts base 10 to base 16.
* **Decimal to Any Base:** Supports conversion to any base from 2 up to 36.

## 🛠️ Installation & Setup
1. Clone the repository:
   git clone [https://github.com/Aldred-Js1/Conversion-of-Numerical-Numbers.git](https://github.com/Aldred-Js1/Conversion-of-Numerical-Numbers.git)
2. Navigate the folder:
   cd Conversion-of-Numerical-Numbers
3. Run any script using python.
